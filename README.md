# nekomaru
This is a TIC TAC TOE game play with cat🐱
Go easy on cat please❤️

## Install
```
$ npm install nekomaru
```

## Usage
start to input `nekomaru`   
You start by input a number.Next, cat turn.Repeat until deicide victory.
